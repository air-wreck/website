But what really is an infinity? Is there such a thing as an "infinite" quantity in the real world? This is a difficult question to answer; we had better retreat to the safe confines of the mathematical world, where "infinity" can be anything that we define it to be.

In the theory of sets, we can think of a set as being finite if there is some natural number n that we can supply such that we can number the elements of the set from 1 to n. For example, the set {apple, orange, banana} is finite; take n=3 to see why.

Infinite sets include ℕ (the set of all natural numbers) and ℝ (the set of all real numbers).

Comparing the Sizes of Sets

I'd like to ask a simple question: when is one set "larger" than another? If the sets involved are finite, then the answer is easy: just let the larger one be the one with more elements. But when the sets are infinite, the answer is not so clear-cut. We intuitively believe that some infinite sets are larger than others. For example, it would seem that there are fewer even integers than there are integers overall, and it would seem that there are more real numbers than natural numbers. However, when dealing with infinite quantities, we must be wary of being led astray by our intuition; in fact, only one of the two statements in the previous setence is correct.
